변경된 파일만 Google Drive 에 동기화합니다.

## 실행 절차

### Step 1 — 동기화 실행

```powershell
rclone sync . gdrive:notes --exclude ".git/**" --exclude ".claude/**" --progress --log-level INFO
```

### Step 2 — 완료 보고

성공 시, 업로드된 파일 목록을 출력한다 (0건이면 "변경된 파일 없음" 표시):
```
☁️ Google Drive 동기화 완료 — gdrive:notes

업로드된 파일:
- 파일명1
- 파일명2
```

오류 발생 시 rclone 오류 메시지를 그대로 출력하고 중단한다.
